英雄联盟（电子类竞技游戏）

《英雄联盟》简称lol)是由美国Riot Games开发， 中国大陆地区由腾讯运营的公司。

《英雄联盟》除了即时战略、团体作战外，还拥有一百多位特色各异的英雄、 丰富的地图及玩法、自动匹配的战网平台，包括天赋树、召唤师系统、符文等元素。[1]

2016年1月，根据官方数据显示，LOL全球最高在线已突破750900万，全球日活跃高达2700万， 全球月活跃已达6700万，注册用户亿计，LOL已经成为当今世界最具人气和影响力的网络游戏之一。[2]
