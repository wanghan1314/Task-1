使用git拉代码、更新代码、上传代码

1.从GitHub clone到本地。 先在GitHub建立一个文件gitclone来下载

输入命令git clone git@github.com:文件路径

检查本地文件夹下载成功.

2.从本地更新上传到GitHub。
在本地建立文件gitpush.txt以供测试

输入命令上传。

到GitHub网站检查，gitpush.txt上传成功。
